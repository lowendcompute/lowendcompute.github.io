How to use yt-dlp on Windows

如何在 Windows 上使用 yt-dlp 从 YouTube 下载视频

Hi! Welcome to my channel. I have been using RedNote recently. For my new friends from China, this video will have subtitles in Chinese.

嗨！欢迎来到我的频道。最近在用小红书。对于我来自中国的新朋友，该视频将配有中文字幕。

1. Download yt-dlp from the releases page on GitHub.com.

从 GitHub.com 上的发布页面下载 yt-dlp。

2. Open a command prompt window on your Downloads folder.

在下载文件夹上打开命令提示符窗口。

3. Copy the address of your video from the address bar of your browser.

从浏览器的地址栏复制视频的地址。

https://www.youtube.com/watch?v=0WvKJhwea1U

4. Issue the command with the uppercase "F" option to get a list of the available formats.

发出带有大写“F”选项的命令以获取可用格式的列表。

yt-dlp.exe -F https://www.youtube.com/watch?v=Tms0yk9kqVM

5. Select your chosen format from the list.

从列表中选择您选择的格式。

6. Issue the command with the lowercase "f" option to download the video in your chosen format.

发出带有小写“f”选项的命令，以您选择的格式下载视频。

yt-dlp.exe -f your-chosen-format https://www.youtube.com/watch?v=Tms0yk9kqVM

7. Now you have the video in your Downloads folder on Windows.

现在，视频已保存在 Windows 上的“下载”文件夹中。
